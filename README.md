# AI-in-Early-Stage-Drug-Discovery-and-Ideation
Exploring the Utility of General-Purpose AI in Early-Stage Drug Discovery and Ideation: A Case Study with ChatGPT


Generate QSAR models.py - python script for generating QSAR models for EGFR and MCL1 activity prediction.

MCL1_Ki_dataset_MACCS.csv - MCL1 dataset, MACCS fingerprints and -log10(Ki) target values. (micromolar)

EGFR_IC50_dataset_MACCS.csv - EGFR dataset, MACCS fingerprints and -log10(IC50) target values. (micromolar)
